# 2026-05-21 07:06 - Documented Four File System Concepts

This session continued concept documentation by creating comprehensive documentation for four file system concepts. These concepts explain how AkashaRender's virtual filesystem works, enabling the stacked directories architecture that is fundamental to plugin extensibility and theme customization.

## Analysis

The following file system concepts were selected as the next priority:

1. **Virtual Filesystem** - How multiple physical directories combine into a unified namespace
2. **File Shadowing** - How files override each other based on mount order
3. **Virtual Paths** - Location-independent file identifiers used throughout the system
4. **MIME Type Detection** - Automatic content type identification from file extensions

These four concepts were prioritized because they describe the VFStack implementation that underpins the stacked directories architecture. Together with the previously documented Stacked Directories concept, they provide a complete understanding of AkashaRender's filesystem virtualization.
