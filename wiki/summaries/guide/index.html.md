# guide/index.html.md - AkashaRender Guide Introduction

## Key Points

- AkashaRender is a rendering engine for creating HTML for websites and EPUB3 electronic books
- Supports nearly any text input format and produces rendered output in various formats
- Uses Markdown or AsciiDoc with YAML frontmatter for content
- Mahabhuta engine enables server-side DOM manipulation with jQuery-like API
- Part of the AkashaCMS ecosystem including plugins and tools

## Summary

This is the introduction to the AkashaRender guide documentation (source: [guide/index.html.md](../../guide/index.html.md)).

AkashaRender is described as "the basis and essence" for converting ideas into websites while maintaining separation between content, layout, and design (source: [guide/index.html.md](../../guide/index.html.md)).

The system is extremely flexible, resulting from nearly 10 years of development and usage. It primarily builds static HTML websites and electronic books, supporting Markdown and AsciiDoc with YAML frontmatter (source: [guide/index.html.md](../../guide/index.html.md)).

Mahabhuta is the DOM manipulation engine that uses a jQuery-like API (provided by Cheerio) allowing plugins to implement transformations and custom tags. This enables powerful features like converting simple tags into complex HTML structures (source: [guide/index.html.md](../../guide/index.html.md)).

Advanced features include LESS to CSS compilation and extensibility through the `package.json` scripts section for integrating external build tools (source: [guide/index.html.md](../../guide/index.html.md)).
